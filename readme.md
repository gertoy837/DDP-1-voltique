# Cara Penggunaan

Buka di terminal
Step 1:
```
.\env\Scripts\activate
```
Step 2:
```
cd .\voltique_app\
```
Step 3:
```
python manage.py runserver
```

## agar tailwindcss nya berjalan
Buka terminal baru
Step 1:
```
cd .\voltique_app\
```
Step 2:
```
npm run build
```

## buka di browser
```
http://127.0.0.1:8000/
```
